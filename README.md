# Get-make-up-exam-topic

主要是用来爬取自己学校补考系统的题目和答案，用来复习补考内容。因为学校补考系统经常上不去，加上考试前会关闭，所以自己写了一个小爬虫，爬取考试数据，用来熬夜复习o(*￣︶￣*)o

运行环境为：vscode编译器，python版本为：Python 3.9.6

运行时，输入学号和考试类型，就会自动抓取考试科目和科目的题目，并保存为html文件到当前目录下。如下图：
